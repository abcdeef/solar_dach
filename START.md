# Startanweisungen

1. Abhängigkeiten installieren:

```bash
npm install
```

2. Server starten:

```bash
npm start
```

3. Im Browser öffnen: http://localhost:3000

Die Anwendung zeigt Eingabefelder für `Breite` und `Höhe` (in Metern). Nach Klick auf "Zeichnen" wird das Dreieck skaliert dargestellt und die berechnete Fläche angezeigt.
